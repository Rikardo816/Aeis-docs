# Seguridad de Dispositivos Móviles

## Objetivos del Módulo
- Comprender los diferentes métodos de conexión de dispositivos móviles.
- Entender los conceptos de gestión de dispositivos móviles (MDM).
- Conocer los enfoques comunes del uso de dispositivos móviles en empresas.
- Entender los riesgos de seguridad y las directrices asociadas con las políticas de uso de dispositivos móviles en empresas.
- Conocer las directrices generales de seguridad y las mejores prácticas para plataformas móviles.
- Comprender las soluciones de gestión de seguridad móvil a nivel empresarial.

## Métodos de Conexión de Dispositivos Móviles
- **NFC:** Comunicación entre dispositivos a menos de 10 cm.
- **Satcom:** Comunicación vía satélite geostacionario.
- **Comunicación Celular:** Basada en torres de red dentro de un radio específico.
- **ANT:** Protocolo de sensor inalámbrico para comunicación entre sensores y controladores.
- **USB:** Comunicación cableada para transmisión de datos y suministro de energía.
- **IR:** Transferencia de datos inalámbrica a corta distancia (hasta 5 m).
- **GPS:** Sistema de navegación y posicionamiento basado en satélites.
- **Wi-Fi:** Tecnología inalámbrica común para hogares y oficinas.
- **Bluetooth:** Comunicación inalámbrica de corto alcance y baja potencia.
- **5G:** Red celular de alta velocidad con baja latencia.
- **P2P:** Comunicación directa entre dos dispositivos sin interferencias.
- **P2MP:** Conexión de un punto a varios puntos.
- **RFID:** Identificación y seguimiento de objetos mediante etiquetas de radiofrecuencia.

## Gestión de Dispositivos Móviles (MDM)
- **Gestión de Aplicaciones Móviles:** Software para proteger datos y aplicaciones móviles.
- **Autenticación Contextual:** Utiliza información del usuario como geolocalización para decisiones de seguridad.
- **Gestión de Contenidos Móviles:** Soluciones para acceder y compartir datos corporativos de forma segura.
- **Gestión de Correo Electrónico Móvil:** Acceso seguro a correos electrónicos corporativos.
- **Seguridad de Dispositivos Móviles:** Medidas para proteger datos y dispositivos móviles.
- **Borrado Remoto:** Técnica para eliminar datos en dispositivos perdidos o robados.
- **Cifrado Completo del Dispositivo:** Cifrado de toda la información almacenada en dispositivos móviles.
- **Geolocalización y Geocercas:** Identificación y monitoreo de la ubicación geográfica de dispositivos.
- **Contenerización:** Separación de datos personales y corporativos en dispositivos móviles.

## Enfoques de Uso de Dispositivos Móviles en Empresas
- **BYOD (Bring Your Own Device):** Empleados usan sus propios dispositivos para acceder a recursos corporativos.
- **COPE (Company Owned, Personally Enabled):** Dispositivos comprados por la empresa, utilizados para trabajo y uso personal.
- **COBO (Company Owned, Business Only):** Dispositivos comprados por la empresa, usados solo para trabajo.
- **CYOD (Choose Your Own Device):** Empleados eligen dispositivos de una lista aprobada por la empresa.

- Desarrollar políticas de seguridad y soporte.

## Riesgos de Seguridad y Directrices
- **Desafíos de Seguridad:** Dispositivos móviles fuera del sistema de seguridad organizacional, más difíciles de rastrear y asegurar.
- **Riesgos Asociados:** Filtración de datos, dispositivos perdidos o robados, falta de actualización de software.
- **Directrices de Seguridad:**
  - **Para Empleados:** Usar canales cifrados, mantener separados los datos personales y laborales, actualizar dispositivos regularmente.
  - **Para Profesionales de Seguridad:** Proteger centros de datos, educar empleados, controlar el acceso basado en necesidad de saber, usar soluciones antivirus y de prevención de pérdida de datos.

## Soluciones de Gestión de Seguridad Móvil a Nivel Empresarial
- **Soluciones MDM:** Despliegue, seguridad, monitoreo y gestión de dispositivos.
- **Soluciones MAM (Gestión de Aplicaciones Móviles):** Gestión de aplicaciones empresariales en dispositivos de empleados.
- **Soluciones MCM (Gestión de Contenidos Móviles):** Acceso seguro a datos corporativos.
- **Soluciones de Defensa contra Amenazas Móviles:** Protegen contra amenazas avanzadas, ataques de red y vulnerabilidades de dispositivos.
- **Soluciones MEM (Gestión de Correo Electrónico Móvil):** Aseguran la infraestructura de correo electrónico corporativo.

## Mejores Prácticas de Seguridad
- **Aplicaciones Móviles:** No guardar contraseñas, usar cifrado, implementar autenticación de dos factores.
- **Datos Móviles:** Cifrar datos, no almacenar contraseñas como contactos, hacer copias de seguridad periódicas.
- **Redes Móviles:** Desactivar interfaces no utilizadas, usar conexiones cifradas, evitar redes Wi-Fi públicas.
- **Plataformas Móviles:** Descargar aplicaciones de tiendas confiables, realizar evaluaciones de seguridad.
