## Fundamentos de IoT
### ¿Qué es IoT?
El Internet de las Cosas (IoT) se refiere a la red de dispositivos con direcciones IP y la capacidad de detectar, recopilar y enviar datos mediante sensores integrados, hardware de comunicación y procesadores.

### Áreas de Aplicación y Dispositivos IoT
- **Industrial:** Dispositivos de manufactura, monitoreo de energía, termostatos.
- **Salud:** Dispositivos de chequeo de salud, dispositivos portátiles.
- **Hogar:** Electrodomésticos inteligentes, cámaras digitales, sistemas de entretenimiento.
- **Transporte:** Vehículos, señales de tráfico, sistemas de peaje.

### Razones para Adoptar IoT
- **Conectividad en Cualquier Lugar:** Conectar dispositivos en cualquier lugar y momento.
- **Automatización y Control:** Facilita la automatización y control de diversos procesos.
- **Monitoreo y Análisis en Tiempo Real:** Permite la recopilación y análisis de datos en tiempo real.

## Arquitectura y Modelos de Comunicación IoT
### Arquitectura IoT
- **Capa de Dispositivos:** Incluye sensores y actuadores.
- **Capa de Comunicación:** Proporciona conectividad a través de gateways e Internet.
- **Capa de Plataforma en la Nube:** Procesamiento y almacenamiento de datos.
- **Capa de Procesos:** Procesos y políticas para gestionar dispositivos y datos.

### Modelos de Comunicación IoT
- **Modelo Dispositivo a Dispositivo:** Comunicación directa entre dispositivos.
- **Modelo Dispositivo a Gateway:** Dispositivos se conectan a través de un gateway.
- **Modelo Dispositivo a Nube:** Dispositivos envían datos directamente a la nube.
- **Modelo Nube a Nube:** Compartición de datos entre diferentes servicios en la nube.

## Seguridad en Entornos Habilitados para IoT
### Consideraciones de Seguridad
- **Transmisión Segura de Datos:** Utilizar cifrado para la comunicación de datos.
- **Gestión de Dispositivos:** Identificación y autenticación de dispositivos.
- **Monitoreo de Seguridad:** Registro y análisis de acciones para detectar comportamientos maliciosos.

### Principios de Seguridad en IoT
- **Capa de Dispositivos:** Implementar autenticación y cifrado en dispositivos.
- **Capa de Comunicación:** Usar protocolos seguros como TLS.
- **Capa de Procesos:** Establecer políticas y prácticas de seguridad.
- **Capa de Nube:** Proteger datos con cifrado y autenticación robusta.

## Mejores Prácticas para la Seguridad IoT
- **Implementar Cifrado de Extremo a Extremo:** Usar Infraestructura de Clave Pública (PKI).
- **Usar VPN:** Para comunicación segura.
- **Desplegar IPS e IDS:** Para detectar y prevenir intrusiones.
- **Bloquear Cuentas por Intentos Fallidos Excesivos:** Prevenir acceso no autorizado.
- **Deshabilitar Cuentas de Invitado y Demo:** Reducir vulnerabilidades.
- **Seguridad Integrada y Unificada:** Implementar un sistema de seguridad integrado.
