# Seguridad de Redes Inalámbricas

## Objetivos del Módulo
- Comprender la terminología, redes y estándares inalámbricos.
- Entender las topologías de redes inalámbricas y la clasificación de redes inalámbricas.
- Conocer los componentes de una red inalámbrica.
- Visión general de los mecanismos de cifrado de redes inalámbricas.
- Entender las medidas de seguridad de redes inalámbricas y herramientas de seguridad inalámbrica.
- Conocer los diferentes métodos de autenticación de redes inalámbricas.

## Fundamentos de Redes Inalámbricas
### Terminología Inalámbrica
- **Ancho de Banda:** Cantidad de información que se puede transmitir.
- **BSSID:** Dirección MAC de un AP que ha configurado un Conjunto de Servicios Básicos (BSS).
- **Hotspot:** Lugar donde hay una red inalámbrica disponible para uso público.
- **Banda ISM:** Conjunto de frecuencias para las comunidades industriales, científicas y médicas.
- **Punto de Acceso (AP):** Conecta dispositivos inalámbricos a una red inalámbrica o cableada.
- **GSM:** Sistema universal para transporte móvil en redes inalámbricas.

### Redes Inalámbricas
- **Ventajas:** Instalación sencilla, acceso desde cualquier lugar dentro del alcance del AP, conexión constante en lugares públicos.
- **Desventajas:** Menor ancho de banda, mayor variabilidad de ancho de banda, tasas de error más altas, menos seguro.

## Tecnologías Inalámbricas
- **Wi-Fi:** Usa el estándar IEEE 802.11 y ondas de radio para la comunicación.
- **WiMAX:** Estándar IEEE 802.16 para redes de larga distancia y alta velocidad.
- **Bluetooth:** Transmite datos entre dispositivos a corta distancia.
- **RFID:** Usa ondas de radio para la identificación y seguimiento de objetos.

## Estándares Inalámbricos
- **802.11 (Wi-Fi):** 2.4 GHz, 22 MHz, 1-2 Mbits/s.
- **802.11a:** 5 GHz, 20 MHz, 6-54 Mbits/s.
- **802.11b:** 2.4 GHz, 22 MHz, 1-11 Mbits/s.
- **802.11g:** 2.4 GHz, 20 MHz, 6-54 Mbits/s.
- **802.11n:** 2.4/5 GHz, 20/40 MHz, hasta 150 Mbits/s.
- **802.11ac:** 5 GHz, 20-160 MHz, hasta 866.7 Mbits/s.
- **802.11ax:** 2.4/5/6 GHz, 40-160 MHz, hasta 10530 Mbits/s.
- **802.15.1 (Bluetooth):** 2.4 GHz, 1-3 Mbps.
- **802.16 (WiMAX):** Estándares para redes de área metropolitana.

## Topologías de Redes Inalámbricas
- **Ad-hoc:** Dispositivos intercambian información sin necesidad de un AP.
- **Infraestructura:** Dispositivos conectados a través de un AP que se conecta a Internet mediante un módem.

### Clasificación de Redes Inalámbricas
- **WLAN:** Conecta usuarios en un área local.
- **WPAN:** Conecta dispositivos alrededor de una persona en un rango muy corto.
- **WMAN:** Usa antenas exteriores para acceder a redes de área amplia.
- **WWAN:** Cubre áreas más grandes como regiones o el globo.

- **Modem Inalámbrico**

- **Pasarelas Inalámbricas**

- **Antenas Direccionales y Omnidireccionales**

## Mecanismos de Cifrado de Redes Inalámbricas
- **WEP:** Utiliza RC4 con IV de 24 bits.
- **WPA:** Utiliza TKIP con RC4 y claves de 128 bits.
- **WPA2:** Utiliza AES-CCMP con claves de 128 bits.
- **WPA3:** Utiliza AES-GCMP 256 y HMAC-SHA-384 para la autenticación.

## Métodos de Autenticación de Redes Inalámbricas
- **Autenticación de Sistema Abierto:** Permite que cualquier dispositivo se autentique con el AP.
- **Autenticación de Clave Compartida:** El dispositivo y el AP usan la misma clave WEP para autenticarse.
- **Autenticación Centralizada con Servidor:** Utiliza un servidor RADIUS para la autenticación EAP.

## Medidas de Seguridad de Redes Inalámbricas
- Crear un inventario de dispositivos inalámbricos.
- Colocar el AP y la antena adecuadamente.
- Deshabilitar la difusión del SSID.
- Seleccionar un modo de cifrado fuerte (preferiblemente WPA3).
- Defenderse contra ataques de crackeo WPA.
- Detectar y localizar APs falsos.
- Configurar la seguridad administrativa en routers inalámbricos.
